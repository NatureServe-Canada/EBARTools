Avertissement : Veuillez consulter notre document sur les méthodes avant d'utiliser CAARBE.
Les données de répartition de CAARBE sont à une échelle relativement grossière et conviennent à des fins de dépistage et d'éducation, mais ne sont pas destinées à tous les types d'applications et d'analyses.
L'absence de données dans certaines zones géographiques ne signifie pas nécessairement qu'une espèce n'est pas présente.
Un écoshape avec une valeur de présence ne signifie pas nécessairement qu'une espèce est présente dans toute la zone géographique.
Définitions de présence : (Veuillez consulter les commentaires ci-dessus pour connaître les exceptions)
Présente - l'espèce est présente dans l'écoshape d'après les données d'observation de l'espèce, les occurrences d'éléments, les caractéristiques sources, l'habitat essentiel fédéral canadien ou l'avis d'experts.
Présence attendue - selon l'avis d'experts, l'espèce pourrait être présente, ou l'écoshape chevauche une estimation de l'aire de répartition ou un modèle d'adéquation de l'habitat.
Historique - toutes les données d'occurrence des espèces dans l'écosystème contiennent des données d'observation datant de plus de 40 ans ou une occurrence d'élément (EO) classée comme éteinte ou historique (classement EO H, H?, X ou X?), ou l'avis d'experts selon lequel l'espèce est éteinte ou historique.
Définitions des types d'utilisation : (Veuillez consulter les commentaires ci-dessus pour connaître les exceptions éventuelles)
Reproduction - l'espèce est considérée comme se reproduisant dans l'écoshape sur la base des codes de reproduction et de comportement eBird ou de l'avis d'experts.
Reproduction possible - l'espèce se reproduit probablement ou peut-être dans l'écoshape sur la base des codes de reproduction et de comportement eBird, BBA ou de la juridiction, ou sur la base de l'avis d'experts.
